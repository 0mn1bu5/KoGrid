Getting Started

Steps for getting started (example on right):

  1. Add references to jQuery and KnockoutJS.
  2. Add references to KoGrid's javascript and css files.
  3. In your html file where you plan to use KoGrid, add:
    <div class="gridStyle" data-bind="koGrid: gridOptions"></div>
    gridOptions is the variable we are going to bind to where we will initialize our grid options.
  4. We are going to define a basic style for our grid in style.css: 
    .gridStyle {
    border: 1px solid rgb(212,212,212);
    width: 400px; 
    height: 300px
}
  5. In your javascript file within the controller where you plan to use
    ng-grid, lets add data that our grid will use:
    6. this.myData = [{name: "Moroni", age: 50},
                 {name: "Tiancum", age: 43},
                 {name: "Jacob", age: 27},
                 {name: "Nephi", age: 29},
                 {name: "Enos", age: 34}];
  6. Now initialize your grid options under the same controller as data:
    this.gridOptions = { data: self.myData };

  7. View the plunker here.


HTML:

<html>

        <title>Getting Started With KoGrid Example</title>

        <script type="text/javascript" src="knockout.js"></script>
        <script type="text/javascript" src="koGrid.js"></script>
        <script type="text/javascript" src="main.js"></script>
    </head>
    <body>
        <div class="gridStyle" data-bind="koGrid: gridOptions"></div>
    </body>
</html>

CSS:

/*style.css*/
.gridStyle {
    border: 1px solid rgb(212,212,212);
    width: 400px; 
    height: 300px
}

Javascript:

// main.js
function mainVm(){
    var self = this;
    this.myData = ko.observableArray([{name: "Moroni", age: 50},
                                      {name: "Tiancum", age: 43},
                                      {name: "Jacob", age: 27},
                                      {name: "Nephi", age: 29},
                                      {name: "Enos", age: 34}]);
    this.gridOptions = { data: self.myData };
};
ko.applyBindings(new mainVm());
